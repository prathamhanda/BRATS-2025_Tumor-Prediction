## Notes

### CUDA Installation
For GPU acceleration, ensure CUDA 11.8+ is installed:
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Alternative CPU Installation
For CPU-only (slower):
```bash
pip install torch torchvision torchaudio
```

### GPU Requirements
- NVIDIA GPU: 4GB+ VRAM minimum
- Tested on: GTX 1650 (4GB VRAM)
- CUDA Compute Capability: 7.5+

### Installation Instructions

```bash
# Create virtual environment
python -m venv ssa_env
source ssa_env/bin/activate  # Windows: ssa_env\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Verify installation
python -c "import torch; print(f'PyTorch {torch.__version__}')"
python -c "import nibabel; print('nibabel OK')"
```

### Optional: Development Environment
For code development and testing:
```bash
pip install jupyter black pylint
```

### Troubleshooting

**CUDA Issues:**
```bash
# Check GPU
nvidia-smi

# Verify PyTorch CUDA support
python -c "import torch; print(torch.cuda.is_available())"
```

**NumPy/SciPy Compatibility:**
If you encounter NumPy version conflicts:
```bash
pip install --upgrade numpy scipy
```

**nibabel Issues:**
```bash
# Reinstall with compatibility
pip install --force-reinstall nibabel==5.1.0
```
